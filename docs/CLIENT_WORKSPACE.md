# Unified Client 360° Workspace

The **Client 360° Workspace** acts as the single pane of glass for Chartered Accountants to manage a client's documents, matching summaries, and checklists.

## Features

- **Split Layout:** Left panel provides overview parameters and health score, Center panel displays tabs for workspace tools, Right panel hosts sidebar checklist tasks, quick notes, and active warnings.
- **Documents tab:** Displays uploaded 26AS, AIS, and TIS sheets.
- **Tax Intelligence tab:** Displays extracted incomes, refund/demand totals, and mismatches.
- **ITR Prep tab:** Progress indicator, missing document checklist, and warnings.
- **Research tab:** Interactively search legislative databases, write workspace notes, and read bookmarks.
- **Timeline tab:** Event logs tracking document completion and task status updates.
